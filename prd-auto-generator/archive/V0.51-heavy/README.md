# V0.51 重量版 存档

> 归档日期：2026-06-15

## 这是哪个版本

prd-auto-generator **V0.51**（模板文件实际标注：version-prd.md V6.0、prd-page-section.md V5.0）。

## 重量版特征

| 维度 | 本版本（重量） |
|------|---------------|
| 组织单位 | **页面级**（每个 .html 一个 6 节章节） |
| 页面章节 | 6 节（功能概述/页面结构/操作流程/字段与交互/业务规则/页面跳转） |
| 字段表 | **11 列**（字段名/标识/类型/必填/数据类型/长度/默认值/校验/取值/来源/错误提示） |
| 全局可选章节 | 4 个（角色与架构/单据流转/接口时序/非功能需求） |
| 版本概述 | 5 子节（页面清单/核心功能/数据来源/统计口径/示例账号） |
| 生成流程 | 13 步 |
| 质量检查 | 7 项 |

## 适用场景（何时回退使用本版）

- 页面交互极度复杂，需要逐 .html 页面详尽描述每个交互元素
- 字段校验规则繁多，需要 11 列全覆盖（含数据类型/长度/默认值）
- 需要独立的"页面结构 section 说明表"逐区域拆解
- 测试/QA 视角的细粒度验收

## 与精简版的区别

| 维度 | 重量版 V0.51 | 精简版（主线） |
|------|-------------|---------------|
| 组织单位 | 页面级 | **模块级** |
| 章节 | 6 节/页 | **3 段/模块**（功能用例与验收/核心流程/数据流向） |
| 字段表 | 11 列 | 8 列（精简但保留校验/错误提示/取值） |
| 编号体系 | RULE-{MODULE}-{SEQ} | **UC/AC-{代号}-{序号}** + 对应用例强绑定 |
| 版本级 | 版本概述 5 子节 | 范围边界 + 角色列表（可选测试账号） |
